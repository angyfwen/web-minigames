How is this educational?

This game is a fun visualization of Boolean Logic, the math that runs computers.

  • Logic Gates: You are acting as physical AND/OR/NOT gates.
  • AND Gate: When code says A AND B, both switches must be ON.
  • OR Gate: When code says A OR B, only one needs to be ON (but both is okay too!).
  • NOT Gate: When code says NOT C, that switch must be OFF.

Analyze the code snippet carefully. The ambulance only moves when the entire equation evaluates to TRUE.
